## 简介
**项目目的：**

1. 基于Python实现一些经典的机器学习算法，加深理解。
2. 归纳总结、转载一些机器学习方面的笔记与文章。

**算法实现主要参考：**

1. 李航《统计学习方法》
2. 周志华《机器学习》
3. 《ESL》

## 数据集
1. 数据集为Kaggle的Titanic数据集：https://www.kaggle.com/c/titanic/data
2. 直接调用处理后的数据集：_fixed.csv
